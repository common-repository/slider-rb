Slider RB - one of the most simple and easiest ways to create an image slider or content slider in your WordPress blog. Simple photo slider block.

== Description ==

Slider RB it's the most simple and easiest way to create an image and photo slider for your WordPress website. Just with a few simple steps, you can create a slider in your Gutenberg page builder.
With the slider RB you can create a slider and manage all slider settings directly in the Gutenberg page builder.
You can upload images to the slider directly in the Gutenberg slider block.
Slider RB is SEO friendly simple image slider that works on all types of devices, including mobile devices.
Simple configuration of the basic slider settings and content transitions effects for the content slider. 
This is one of the simplest photo slider solutions with the slider block mode which helps to create a slider with photos in Gutenberg editor in a few simple steps. 

Which modes are available in Slider RB: image slider, photo slider, slider block, content slider, video slider

Basic Features:

- Slider height
- Slider width
- Slider unit height
- Slider unit width
- Slider animation direction
- Autoplay mode
- Timer animation delay
